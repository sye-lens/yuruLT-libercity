# yuruLT-libercity

このリポジトリは、ゆるゆるLT会の告知分などを管理するために作成しました

# 使い方

1. このリポジトリを`git clone`する
2. ローカルのエディタでブランチを作成、markdownで告知文、概要欄を編集します。
3. PRを作ってみんなのチェックをもらいます
4. チェックをもらったら`main`ブランチにmergeします
5. 新しく告知文や概要欄を作成したり、変更を加えるには、`git fetch origin`,`git switch main`, `git merge origin/main`の順でコマンドを実行し、自分のローカルにローカルの最新状態を反映させてから編集を開始します。
