# yuruLT-libercity

このリポジトリは、ゆるゆるLT会の主催をより楽にしたり、会の内容を管理するために作成しました

# 前提

- Githubアカウント
- Gitの基本的な操作ができること

# Tutorial

0. 管理者である`sye-lens`にGithubアカウントを伝える
1. このリポジトリを`git clone`する

# 編集方法

1. 新しく告知文や概要欄を作成したり、変更を加えるには、以下の手順でコマンドを実行し、自分のローカルにリモートの最新状態を反映させます

```
git fetch origin
git switch main
git merge origin/main
```

2. 最新状態をローカルに取り込んだら、新しくブランチを切って編集を開始します
3. 編集が完了したら以下の手順でリモートに`push`します。

```
git add （編集したファイル名）
git commit -m "（適当なコミットメッセージ）"
git push origin （ブランチ名）
```

4. PRを作ってみんなのチェックをもらいます(チェックをしてほしい場合)
5. チェックをもらったら`main`ブランチにmergeします

# 注意点

- リベのメッセージ機能の部分では`markdown`が使用できません。なので、エディタでは必ずプレーンテキストで文章を作成してください。
